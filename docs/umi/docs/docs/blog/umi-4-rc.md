### Umi 4 RC 发布的新功能有哪些？

Umi 4 RC发布了以下新功能：

- 新官网和文档
- MFSU V3 & 默认开启
- 双构建引擎和ESMi
- Webpack 5
- React Router 6 & 新路由
- 最佳实践迭代
- 依赖预打包
- Umi Max
- Low Import 研发模式
- 强约束功能集成
- Import All From Umi 迭代
- srcTranspiler 和 depTranspiler
- jsMinifier 和 cssMinifier
- 应用元数据
- 微生成器
- 贴心小改进

还有一些计划在正式版发布之前做的事情，包括api route、umi server and adapter、route loader、稳定的 lint、更多命令、组件研发 father 4、文档工具 dumi 2等。
